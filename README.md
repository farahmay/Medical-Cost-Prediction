# Medical-Cost-Prediction
Machine Learning project for predicting medical charges

## Description
This project aims to predict medical insurance charges using Machine Learning models.

## Models Used
- Linear Regression
- Ridge
- Lasso
- ElasticNet

## Features
- age
- bmi
- smoker
- children
  
## Results
Linear Regression was identified as the best model.
